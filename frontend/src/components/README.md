# components/

This directory contains **shared, reusable React components** used across multiple pages.

## Planned Components (Phase 5 - UI Module)

### ui/
- `Button.jsx` — Polymorphic button with variants (primary, secondary, danger, ghost)
- `Input.jsx` — Controlled input with label and error display
- `Badge.jsx` — Status and priority badges with color maps
- `Modal.jsx` — Accessible modal with focus trapping
- `Spinner.jsx` — Loading spinner
- `Avatar.jsx` — User avatar with initials fallback
- `EmptyState.jsx` — Empty list placeholder with icon and CTA

### complaint/
- `ComplaintCard.jsx` — Complaint summary card
- `ComplaintStatusBadge.jsx` — Color-coded status badge
- `ComplaintPriorityBadge.jsx` — Color-coded priority badge

### notification/
- `NotificationBell.jsx` — Navbar notification icon with unread count
- `NotificationItem.jsx` — Single notification row

### layout/
- `Sidebar.jsx` — Role-aware navigation sidebar
- `Navbar.jsx` — Top navigation bar with user menu
